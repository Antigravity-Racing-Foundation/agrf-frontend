◾ Project Info

Welcome to the official Antigravity Racing Foundation Website!

Our flagship project, Thallium+Beat, allows you to play WipEout HD and WipEout Pulse online once more, as well as re-experience most of 2048's online features.

Currently,

  • WipEout HD has full feature parity with original servers;
  • WipEout Pulse has every multiplayer feature implemented and a recreation of the wipeout-game.com website is available (but it's missing leaderboards, team allegiance and ghost sharing);
  • WipEout 2048 has full feature parity with original servers except for Online Campaign, so the only way to play online for now is HD Cross-Play.

We hold several weekly events on our WipEout Discord server, including:

  • WipEout HD Weekly Lobby, hours of fun every Saturday at 19:00 UTC
  • WipEout HD Mode Shuffle, an appreciation of often neglected ways to enjoy the game every Sunday at 19:00 UTC
  • WipEout Pulse Portable Rush, its version of Weekly Lobby, held every Thursday at 19:00 UTC

If you have any questions, don't hesitate to reach out to us! (@thatonebonk, @chacheechoo on Discord)
Thanks, and have fun!

P.S. Thallium also supports MotorStorm Arctic Edge, everything works except Community (friends list and block list) features, which relied on a deprecated PSN API. Additionally, you can join the MS World Online Discord to find other MotorStorm players to play online with!

◾ Setup Guide

Getting started

If your PS3, PS Vita or PSP system already uses PS Rewired's DNS, then you're all set! We've partnered with them so that their DNS redirects to us.
If not, then to start playing WipEout HD, WipEout 2048, WipEout Pulse or MotorStorm Arctic Edge online, you will have to configure your system to use Thallium/Beat.

PlayStation 3 system

This process won't take longer than a minute. To connect to Thallium:

  • Go to the Settings tab in the XMB
  • Go down to Network Settings
  • Pick Internet Connection Settings
  • Confirm to start configuring
  • On the Select a setting method screen, pick Custom
  • Continue the setup according to your preference, picking the default options whenever you are unsure until you reach the DNS Setting screen
  • Select Manual
  • Set Primary DNS to 147.135.213.57 and the Secondary DNS to 1.1.1.1
  • Continue the setup as before and save your settings
  • Testing connection is optional
  • You should be done

PlayStation Vita system

To connect to Thallium/Beat using a Vita:

  • Go to Settings
  • Select Network
  • Select Wi-Fi Settings
  • Pick your desired network
  • Pick Advanced Settings
  • Set DNS Settings to Manual
  • Set Primary DNS to 147.135.213.57 and the Secondary DNS to 1.1.1.1
  • Click OK
  • Quit Settings
  • You should be done
  • Note that the Online Pass is required to play 2048 online
  • You should be able to access Beat through Adrenaline now; if not, follow the PSP system setup guide through Adrenaline's XMB

PlayStation Portable system

To connect your PSP to your Wi-Fi network, you can either install the wpa2psp plugin, install/update ARK-4 CFW (which will include wpa2psp (recommended)), or figure it out yourself.

You must be on firmware version 6.61 and signed into PlayStation Network on your PSP to play online.

To connect to Beat:

  • Go to the Settings tab in the XMB
  • Pick Infrastructure Mode
  • Select a connection to edit or make a new one
  • Continue the network setup until you reach DNS Settings
  • Pick Manual
  • Set Primary DNS to 147.135.213.57 and the Secondary DNS to 1.1.1.1
  • Continue the setup according to your liking
  • Testing connection is optional
  • You should be done
  • For ease of use, when you are in-game, go to Multiplayer -> Online Account and enable Auto-Login

RPCS3 Emulator

To configure RPCS3 to use our Thallium backend, follow this guide: https://wiki.rpcs3.net/index.php?title=Help:Netplay#Global_configuration

The IP/Host Switches you need is this string:

wipeout2048.muis.online.scee.com=147.135.213.57&&wipeout2048.online.scee.com=147.135.213.57

PPSSPP Emulator

PPSSPP versions 1.19 and above connect to WipEout Pulse and MotorStorm Arctic Edge online automatically!

  • Install/update the emulator
  • Check Settings -> Networking -> (Networking category) -> Enable networking/WLAN
  • Confirm your username in Settings -> Networking -> (Infrastructure category) -> Username
  • You should be done

If you're having trouble connecting, try disabling Settings -> Networking -> (Infrastructure category) -> Autoconfigure and setting the DNS server value to 147.135.213.57

Creating a WipEout Pulse ship skin

Beat supports the WipEout Pulse custom ship skins feature. With it, you can create a custom ship skin and bring it online!

To create a custom WipEout Pulse ship skin:

  • Download Thomas Perl's (aka thp) shipedit utility to create your skin, available for Linux and Windows
  • Export your custom skin
  • Send @chacheechoo the following information in Discord DM's:
    • The "16034453" file in the exported skin's folder;
    • Your PSN/PPSSPP Username;
    • The slot that you want to use (0-4);
    • The name of your custom skin;
    • Whether or not you would like the skin to be potentially publicly available to other players in the future
    • Note that your skin may take a while to be patched, or we could refuse to patch it
  • You will be given a patched skin - unpack the archive and put the UCES00465DTEAMSKIN000X folder in [PSP storage root]/PSP/SAVEDATA/
  • Check that the [Multiplayer...] -> [Online Account] -> [Ship Skins] option is on

You should be done

We are not responsible for any issues that may arise when using custom skins on any platform


Now, you should be done. Troubleshooting steps are available on our Discord server, if you require them.
